Check network requests

Every imported file becomes a separate network request. From one side we have automatic lazy loading because we don't have to split our bundle with some webpack loadable plugins. But from the other hand we would still need to at list minify our files for production and we wouldn't want every single file to be 'lazy-loaded' we would want to combine some part of the logic together in one bundle. So it turns out webpack is still needed even if we don't care about older browsers support.
